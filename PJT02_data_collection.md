주피터 노트북 사용: vs code, 코랩, 주피터노트북 설치 등  

## 데이터 사이언스
데이터 사이언스: 수 많은 데이터를 모아서 필요한 정보만을 이용해 내가 원하는 정보 혹은 새로운 지식을 알아내는 과학 분야  
1. 수학 및 통계 지식
2. 해킹 실력
3. 도메인 전문지식(금융, 로봇 등 자신이 목표하는 분야의 지식)

### 데이터 사이언스 프로세스
1. 문제 정의: 해결하고자 하는 문제 정의
   - ex) 구글의 주식 가격은 앞으로 어떻게 될까?
2. 데이터 수집: 문제 해결에 필요한 데이터 수집
   - 웹 스크래핑
   - 웹 크롤링
   - Open API
   - **캐글**, Data.world, 데이콘(공모전), 공공데이터포털
3. 데이터 전처리: 실질적인 분석을 수행하기 위해 데이터를 가공하는 단계
   - 불완전하거나 오류가 있는 데이터 제거하여 데이터의 품질 개선
   - 중복 데이터 제거
   - 데이터 변환
   - Numpy
     - 다차원 행렬 구조를 제공하고 행렬 연산이 빠르다
   - Pandas
     - 시리즈(1차원), 데이터프레임(2차원), 피벗 등
4. 데이터 분석: 전처리가 완료된 데이터에서 필요한 정보를 추출하는 단계
5. 결과 해석 및 공유: 의사 결정에 활용하기 위해 결과를 해석하고 시각화 후 공유하는 단계  


  ### CSV?
  몇 가지 필드를 쉼표(,)로 구분한 텍스트 데이터 및 텍스트 파일
  표 형식의 데이터를 CSV 형태로 많이 사용
